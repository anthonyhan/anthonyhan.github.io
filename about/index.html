blog.log

2005-10-14：开通 MSN Spaces，首次尝试网络博客；
2006-07-14：百度推出Baidu空间，由于教育网内访问速度快，还有搬家功能，遂迁移至此,偶尔使用；
2007-10-18：MSN Spaces 被升级成 Windows Live Spaces；
2008-07-29：停用 Baidu空间，只保留 Live Space；
2010-09-28：微软宣布关闭博客服务，所有用户将导入到 wordpress.com；
2010-12-10：被迫转到 wordpress.com，Windows Live Spaces 将于2011年3月关闭；
2011-01-28：无法忍受 wordpress.com 的缓慢、自定义功能非常弱、而且经常“连接被重置”，果断申请了域名和空间；
2014-04-24：WP更新至v3.9，换下服役3年的 simple-dark 主题，移除静态Cache插件，增加duoshuo，Crayon Syntax Highlighter 等；
2015-02-01：重新安装WP4.1，移除冗余插件。荒废许久，准备更新。
2016-08-16：启用Hexo，整理归档内容，准备逐步迁移到Hexo。
2016-11-29：启用新域名 www.limbonova.com。
2017-04-10：多说评论将停止服务，切换至搜狐畅言评论系统。
2017-09-15：更换评论系统到 HyperComments 。
2018-03-07：启用 https 。
2018-10-29：Updated Hexo to 3.8.0, hexo-next-theme to 6.4.2, removed HyperComments.
2018-12-08：updated next-theme to 6.6.0 and added gitalk.
